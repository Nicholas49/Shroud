 -- Shroud --

1. What it is

Shroud is a simple python program that allows users to securely encrypt messages for sending back and fourth. The messages are encrypted using a substitution cipher that is essentially unbreakable without knowledge of the user's password.
Although I'm no proffessional I've worked pretty extensively to make sure this program is cryptographically secure and without major exploitable vulnerabilities.
That being said, I can't recommend using this for sending any highly sensitive data. It's a project developed by an amateur, not a professional program, so treat it as such.
